# ws2_32 -- MicroNT Windows Sockets

Flat BSD sockets over AFD, plus the Winsock 2 entry points modern toolchains
(Rust std, Go, ...) link.  This SDK ships only winsock.h (1.1); the 2.x type
surface lives in `winsk2p.h`.

## Implemented

- BSD 1.1 core (socket/bind/connect/listen/accept/recv/send/select/...).
- Synchronous WSA 2 data path: `WSASocketW`, `WSARecv`, `WSASend`, `WSAIoctl`.
- Name resolution: `getaddrinfo` / `freeaddrinfo` over the hosts-file resolver.

## Not implemented (truthful errors, not stubs)

- `WSADuplicateSocketW` -> `WSAEOPNOTSUPP`.  Cross-process socket sharing needs
  an AFD socket-duplication path that does not exist yet; callers are told the
  operation is unsupported rather than handed an unusable protocol-info blob.
- Overlapped (async) socket I/O (the `lpOverlapped != NULL` path of WSARecv/
  WSASend) is the async/IOCP milestone; the synchronous path (which covers
  blocking std::net) is real.  See MODERN-IPSTACK.md.
